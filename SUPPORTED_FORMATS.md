- **Web**: Use WebP or optimized JPG
- **Print**: Use TIFF or high-res PDF
- **Icons**: Use ICO or SVG

### For Audio
- **Music**: Use FLAC (archive), MP3 320kbps (portable)
- **Podcasts**: Use MP3 128kbps
- **Audiobooks**: Use MP3 64kbps mono
- **Professional**: Use WAV

### For Video
- **Web**: Use MP4 (H.264) or WebM
- **Archive**: Use MKV (H.265)
- **Compatibility**: Use MP4
- **High Quality**: Use MKV with HEVC

### For Archives
- **Maximum Compression**: Use 7Z
- **Compatibility**: Use ZIP
- **Unix**: Use TAR.GZ
